# Multi-Objective-Optimization-of-Steel

[**English README**](README.md) | [**日本語版 README**](readme_jp.md) 

## 项目概述

本仓库提供了以下论文的官方代码实现：

“Multi-Objective Optimization of Corrosion Resistance, Strength, Ductility Properties of Weathering Steel Utilizing Interpretable Attention-Based Deep Learning Model” (发表于 npj Materials Degradation, https://doi.org/10.1038/s41529-025-00654-y)

本仓库包含以下内容：

- `feature_select.py`：使用注意力模型从 xlsx 文件中提取特征。
- `feature_sort_attention.py`：用于特征排序。
- `mlp_predition_all.py`：用于预测满足第一作者要求的特征。

第一作者根据预测结果选择了两种成分用于炼钢，并进行了一系列实验。
