## プロジェクト概要 (Overview)

本リポジトリは、以下の論文の公式実装を提供するものです：

“Multi-Objective Optimization of Corrosion Resistance, Strength, Ductility Properties of Weathering Steel Utilizing Interpretable Attention-Based Deep Learning Model” (npj Materials Degradation 掲載, https://doi.org/10.1038/s41529-025-00654-y)

本リポジトリには以下のファイルが含まれます：

- `feature_select.py`: アテンションモデルを用いて xlsx ファイルから特徴量を抽出
- `feature_sort_attention.py`: 特徴量のソート用
- `mlp_predition_all.py`: 筆頭著者の要求を満たす特徴量の予測用

筆頭著者はこれらの予測結果に基づき、製鋼に使用する2つの成分を選択し、一連の実験を行いました。
